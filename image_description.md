## 注意事项

1. 所有图标应当保持同一风格，推荐使用扁平化设计
2. 图标最好提供2x和3x尺寸，以适应不同分辨率的设备
3. 图片格式优先使用PNG格式，保证透明背景和清晰度
4. 较大的图片(如banner)可考虑使用JPG格式以减小文件体积
5. 确保所有图片资源版权合规
